# Claw Logs

> 📊 Centralized logging & observability for OpenClaw instances.

[![CI](https://github.com/mxn2020/claw-logs/actions/workflows/ci.yml/badge.svg)](https://github.com/mxn2020/claw-logs/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Real-time log streaming, dashboards, alerts, and instance monitoring for your OpenClaw deployment.

## ✨ Features

- **🖥️ Live Log Stream** — Real-time log viewer with search, level filters, pause/resume
- **📈 Dashboard** — Log volume charts, error rates, latency metrics
- **🔔 Alerts** — Configurable alert rules for errors, rate limits, instance health
- **🖧 Instance Monitoring** — CPU, RAM, disk usage, agent counts per instance
- **🌗 Dark & Light Mode** — Terminal-inspired dark theme + light alternative
- **⚡ Real-time** — Powered by Convex for instant updates

## 🚀 Quick Start

```bash
git clone https://github.com/mxn2020/claw-logs.git
cd claw-logs
npm install
npm run dev
```

## 🧪 Testing

```bash
npx vitest run
```

## 🛠️ Tech Stack

React 19 · TypeScript · Vite · Convex · JetBrains Mono · Vitest
